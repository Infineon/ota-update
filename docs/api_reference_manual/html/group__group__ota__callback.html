typedef cy_rslt_t( * cy_ota_agent_mem_read)(cy_ota_mem_type_t mem_type, uint32_t addr, void *data, size_t len)

Memory read callback function pointer type.

Callback of this type is invoked when a OTA Agent wants to read from memory.

A function of this type must be implemented by the application to read contents from given memory location.

Parameters
[in]mem_typeThis is of type cy_ota_mem_type_t. Application can use this to implement read operation functionality.
[in]addrStarting address to read from.
[out]dataPointer to the buffer to store the data read from the memory.
[in]lenNumber of data bytes to read.
Returns
CY_RSLT_SUCCESS CY_RSLT_TYPE_ERROR
